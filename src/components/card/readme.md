#### Usage

How to install, instantiate, or embed the card components.

##### Global Script

    <!-- Global Script -->
    <script src="dist/js/patterns.js"></script>

    <script>
      var Patterns = new Patterns();
      var card = Patterns.card();
    </script>

##### Module Import

    import Card from 'src/components/card/card';

    new Card();
