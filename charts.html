Line Chart

To achieve a smooth line, set the following properties in the dataset object:

pointBackgroundColor: 'transparent',
pointBackgroundColor: 'transparent',
pointBorderColor: 'transparent',
lineTension: 0,

Additional configuration options are viewable on Chart.js .
